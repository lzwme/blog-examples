# ghproxy 可用站点列表

收集可用于 github 附件代理下载的 `ghproxy` 站点列表，并每日自动更新可用性。

## 自建代理镜像

选择以下方案之一，参考其指引文档进行搭建。

- 基于 [hunshcn/gh-proxy](https://github.com/hunshcn/gh-proxy) github release、archive以及项目文件的加速项目
- 基于 [0-RTT/ghproxy-go](https://github.com/0-RTT/ghproxy-go) 基于Go语言开发的Github文件代理加速工具
- 基于 [moeyy01/gh-proxy-go](https://github.com/moeyy01/gh-proxy-go) gh-proxy 的 Go 语言版本
- 基于 [WJQSERVER-STUDIO/ghproxy](https://github.com/WJQSERVER-STUDIO/ghproxy) 基于Go的高效&多功能的GHProxy
- 基于 [sky22333/hub-proxy](https://github.com/sky22333/hub-proxy) 自建Docker镜像仓库和Github加速

## 其他参考

- [Philanthropy](https://uptime.akams.cn/status/philanthropy)
- API: https://api.akams.cn/github
