必应每日壁纸API - 用法示例

      // 返回第一张图片，并重定向到图片地址
      https://lzw.me/x/iapi/bing/api.php

      // `json`: 指定返回为 json 信息
      https://lzw.me/x/iapi/bing/api.php?json=1

      // `n` 指定返回数量。默认是 1，最多返回 8 条。若大于 1，返回图片从结果列表中随机取
      https://lzw.me/x/iapi/bing/api.php?n=8
      https://lzw.me/x/iapi/bing/api.php?json=1&n=8

      // `idx` 请求图片截止天数(-1 明天，0 今天，1 昨天，2 前天， 最多为 7，即 8 天前)。默认是 0
      https://lzw.me/x/iapi/bing/api.php?json=1&n=8&idx=1
